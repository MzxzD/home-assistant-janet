## 🚀 Installation for Users

### **Option 1: Manual Installation**
```bash
cd /config/custom_components
git clone https://github.com/MzxzD/home-assistant-janet.git janet
mv janet/custom_components/janet .
rm -rf janet
```

### **Option 2: HACS (Future)**
Once submitted to HACS:
1. HACS → Integrations → Custom Repositories
2. Add: `https://github.com/MzxzD/home-assistant-janet`
3. Category: Integration
4. Install "Janet AI Companion"
